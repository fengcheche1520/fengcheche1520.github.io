在ubuntu中利用hexo和github搭建博客

发表于 2016-11-20   |  

利用hexo框架和github搭建博客主要分两个步骤：

  1. 在本地搭建好博客。（就是配置好写博客的环境）
  2. 将本地博客部署到远程服务器github上。（让本地的博客与远程服务器关联起来）

第一部分 博客环境配置

  • 安装nodejs

hexo基于nodejs，利用nodejs生成文章

  • 安装npm

npm是hexo的安装工具

  • 创建blog目录

blog目录就是以后写博客的工作目录

  • 安装hexo

错误解决：
node：not found
添加链接ln -s /usr/bin/nodejs /usr/bin/node

  • 初始化本地环境

  • 生成本地页面并预览

在浏览器中输入localhost:4000预览博客

执行完hexo g后会在source/_posts/目录下生成xxx.md文件，这里存放了所有的博客文章
hexo s（开启预览访问端口，在浏览器中输入localhost:4000预览，指定-p可以自定义端口，如hexo s -p 12345）

第二部分 创建github仓库，部署本地博客到github

  • 注册github帐号并创建仓库（repository）

    注意：xxx要和github的用户名相同

  • 修改本地hexo配置
    用vim打开_config.xml

  • 配置git用户信息并安装部署工具

  • 部署本地博客到github

将.deploy目录部署到github，在浏览器中输入http://xxxx.github.io.git访问

  • 一些常用的命令:
    hexo c 清除生成的静态网页
    hexo g 生成静态网页
    hexo d 将本地的静态页面部署到github上
    hexo n “acticlename” 新建文章
    hexo n page “pagename” 新建页面
    hexo s -g 先生成静态页面然后开启本地预览
    hexo d -g 生成静态页面然后部署到github
